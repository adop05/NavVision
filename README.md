# NavVision

A web application using LiDAR, Mapbox Open Source map API, JavaScript, CSS, and HTML to help visually impaired individuals navigate new and known experiences with confidence independently.

We use the LiDAR sensor on a phone (most likely an iPhone) for object detection which will send a signal that produces a audible noise instantaneously that increases in repetition the closer one gets to the object. Mapbox turns map coordinates from their source into text and then grabs the user's coordinates and calculates a walkable route to the destination coordinates.
